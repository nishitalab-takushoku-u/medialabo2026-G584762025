第一回WWW と HTML

今回の講義では、WWW HTMLについて学んだ、URLにはプロトコル(https:)、サーバ、パスがあり、これらがインターネット上の膨大なデータからWebページを探すのに役立っている。HTMLではレイアウトを変えるためにたくさんのタグがある。 <p></p>ではタグ内の文が一つの要素となり<strong></strong>ではタグ内の文字が強調される <img src="〜〜">では〜〜に画像ファイルを入力することで画像が表示される。画像ファイルの後ろにalt="〜〜"を入力すると閲覧者が読み上げソフトを使用していると〜〜内にある文字が読み上げられる。ul は番号なしのリストを作る、ol は番号付きのリストを作るなど学んだ。

演習の成果:演習1-4

第二回 CSS による Web ページの装飾

今回の講義では、主にCSSについて学んだhtmlにcssファイルをリンクさせるためには、<link rel="stylesheet" href="[cssファイル名]"≶を使用しcssファイルに変更したい内容を書き込んでいく。 まず変更したいタグを決めて[タグ名]{}の中に記入していく。coler:[色]で文字のいろの指定し、 text-shadow:で文字に影がつく。borderで文字の周りに線を敷くことができ、border-[線を引く場所]-style:[線の種類]で線の引き方・線の種類（例：棒線solid、点線dashed　など）を設定できる。 backgroundでは背景の色を指定できlinear-gradientでグラデーションをつけることができる。文章から距離を作りたいときは、margin(線と前の要素との距離)、padding(線と文章の距離)を使うなど学んだ。
